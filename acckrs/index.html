About this page

Hi and welcome to the first version of my portfolio page.

This page is created with the following user stories:

  • User Story 1: I can access all of the portfolio webpage's content just by scrolling.
  • User Story 2: I can click different buttons that will take me to the portfolio creator's different social media pages.
  • User Story 3: I can see thumbnail images of different projects the portfolio creator has built.
  • User Story 4: I navigate to different sections of the webpage by clicking buttons in the navigation..
  • User Story 5: I can see local weather information in Celsius and Fahrenheit degrees..
  • User Story 6: I can load quotes and their authors using ajax api and tweet them if I like..
  • User Story 7: I can search Wikipedia using and get random Wikipedia article..
  • User Story 8: I can check TwichTV streamers and see if they are online..
